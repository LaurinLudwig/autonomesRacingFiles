# Projektübersicht & Nützliche Links

Dieses Repository enthält alle relevanten Unterlagen, Diagramme und Dateien rund um die Entwicklung unserer mobilen Roboterplattform und der zugehörigen Software-/Systemarchitektur.

## Inhalt des Repositories

- **CAD/**  
  enthält .Step und .3MF Dateien zum Drucken von Ersatzteilen

- **ChangedRosFiles/**  
  angepasste ROS-Dateien für unsere Plattform

- **Finanzplan_Antrag_Roboter_plattformen.xlsx**  
  Auflistung der verwendeten/bestellten Komponenten

- **KabelDiagramme.pdf / KabelDiagramme.pptx**  
  Übersicht der Verkabelung, Schnittstellen und elektrischen Verbindungen

- **SystemDiagramme.pdf / SystemDiagramme.pptx**  
  Schaubild der Komponenten

- **ReadmeAndUsefulLinks.md**  
  (Diese Datei) Kurze Einführung in das Repo und Sammlung relevanter Ressourcen.
